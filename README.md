# 📊 Amazon Sales Data Analysis Dashboard

## 🔍 Project Overview
This project focuses on analyzing Amazon sales data to uncover key business insights such as sales trends, customer behavior, product performance, and operational efficiency.

The analysis was performed using **SQL for data extraction** and **Excel for visualization and dashboard creation**.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing regions and categories  
- Study customer buying patterns  
- Evaluate delivery and fulfilment efficiency  
- Build an interactive dashboard for insights  

---

## 📂 Dataset Description
- Total Records: 1,20,378  
- Key Features:
  - Order ID  
  - Date  
  - Order Status  
  - Fulfilment Type  
  - Category  
  - Size  
  - Quantity (Qty)  
  - Amount  
  - State & City  
  - B2B Indicator  

---

## 🛠 Tools & Technologies
- **Excel** → Data Cleaning, Pivot Tables, Dashboard  
- **SQL (MySQL)** → Data Analysis

## 🔑 Key Insights

### 🗺 Top Performing Regions
Kerala is the highest revenue-generating state, followed by Maharashtra and Uttar Pradesh. Sales are concentrated in a few key regions.

### 📈 Seasonal Sales Trend
Sales peak in March, indicating strong seasonality. After that, sales decline and stabilize at lower levels.

### 🥧 Order Status
Most orders are successfully shipped, but a noticeable portion is cancelled, indicating improvement opportunities.

### 🚚 Fulfilment Efficiency
Amazon fulfilment handles more orders and performs better than merchant fulfilment.

### 📊 Category Performance
A few categories like Set and Western Dress dominate overall sales.

### 👕 Size Analysis
Medium and Large sizes are most in demand, while extreme sizes show lower demand.

---

## 📌 Conclusion
Sales performance is influenced by seasonal trends, regional demand, and product preferences. Operational improvements can enhance efficiency and profitability.
